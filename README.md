# TacticalBoard v3

Interface de trading tactique complète — deux comptes indépendants (CFD Chartiste + Options Sèches), squeeze scanner connecté, grecques options, analyse technique.

## Stack
- React 19 + Vite
- Recharts (graphiques)
- Lucide React (icônes)
- Tailwind CSS (CDN)

## Déploiement automatique
GitHub Pages via GitHub Actions — chaque push sur `main` déclenche un build + déploiement.

## Dev local
```bash
npm install
npm run dev
```
